# Kingpin

AI-native modular business OS for SMBs (1-50). ERP + CRM + capability management on a shared, human-and-AI-readable Company Info Layer. Any AI plugs in: managed, BYO-key, or self-hosted — via MCP.

## Working rules
1. Every plan, decision, architecture change, and iteration is documented before/with the code
2. Decisions get an ID (D/A/I/M/B-prefix) and land in the Decision Register
3. Effective communication — no long stories
